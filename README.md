# IVF PhD Topic Discovery

This workspace is for systematic PhD topic discovery in AI-enabled IVF research.

The documentation site is built with MkDocs and is organized around the research workflow:

- evidence base and paper registry
- literature review matrix
- gap frequency analysis
- theme classification
- research problem development
- research questions, objectives, and candidate topics

## Build

```bash
mkdocs build --strict
```

## Serve Locally

```bash
mkdocs serve
```

Open:

```text
http://127.0.0.1:8000
```
