# Source Quality Register

This page tracks which sources are safest to use in presentation.

## High-Confidence Sources

| Source | Use For |
| --- | --- |
| [WHO infertility fact sheet](https://www.who.int/news-room/fact-sheets/detail/infertility) | Global infertility burden |
| [WHO 2023 infertility news release](https://www.who.int/news/item/04-04-2023-1-in-6-people-globally-affected-by-infertility) | 1 in 6 global infertility statistic |
| [HFEA 2023 fertility treatment trends](https://www.hfea.gov.uk/about-us/publications/research-and-data/fertility-treatment-2023-trends-and-figures/) | UK IVF success by age |
| [CDC National ART Summary](https://www.cdc.gov/art/php/national-summary/index.html) | US ART age and live-birth summary |
| [ESHRE ART fact sheet 2025](https://www.eshre.eu/-/media/sitecore-files/Press-room/ESHRE_ARTFactSheet_v10_2025.pdf) | Europe/global ART benchmark rates |
| [NFHS-5 overview](https://microdata.worldbank.org/index.php/catalog/4482) | India fertility and health survey context |
| [HTAIn IVF cost policy brief](https://htain.dhr.gov.in/images/pdf/Policy_Briefs/79.Policy_Brief_IVF.pdf) | India IVF cost and catastrophic expenditure |
| [National ART & Surrogacy Registry](https://registry.artsurrogacy.gov.in/) | Indian ART registry context |

## Medium-Confidence Sources

| Source | Use For |
| --- | --- |
| Peer-reviewed Indian infertility prevalence papers | India infertility burden |
| ART access studies in LMICs | Affordability and access discussion |
| Frontiers fertility decline review | Lifestyle and social drivers |

## Use Carefully

| Source Type | Why Careful |
| --- | --- |
| Market reports | Useful for market size, but may be commercial. |
| Clinic websites | Useful for local package examples, but not neutral evidence. |
| News articles | Good for context, not core academic evidence. |
| Social media | Avoid for formal presentation. |

## Presentation Rule

For every number, ask:

1. What is the source?
2. What year is it?
3. What population does it describe?
4. What denominator is used?
5. Is it India-specific or international?
