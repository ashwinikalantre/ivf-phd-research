# Context and Statistics Evidence Pack

This section prepares answers for university faculty panel questions.

These questions are different from ML literature questions. They ask:

- Why IVF?
- How big is the problem?
- How successful is IVF?
- How does age affect success?
- What is the Indian context?
- What is the economic burden?
- How does lifestyle matter?
- Why is decision support needed?

## Evidence Rule

Use only confident sources for presentation.

| Source Type | Confidence |
| --- | --- |
| WHO, CDC, HFEA, ESHRE, NFHS, Government of India | High |
| Peer-reviewed journal articles | High/Medium |
| Government registry/official portal | High |
| Market reports | Medium/Low, use only for market size/demand context |
| Clinic blogs | Low, avoid unless discussing local cost examples |

## Key Talking Point

The PhD is not justified only because IVF exists.

It is justified because IVF is:

- medically important
- emotionally and financially demanding
- age-sensitive
- outcome-uncertain
- data-rich
- increasingly dependent on personalized decisions

This makes explainable clinical decision support relevant.
