# Repeated Gaps Only

This page filters the gap list into gaps that currently look PhD-relevant.

## Strong Gaps

| Gap | Why It Matters |
| --- | --- |
| Need external validation | Many models are trained and tested in one country, clinic or dataset. Without external validation, clinical use is weak. |
| Need clinical decision support | Many studies stop at prediction accuracy. A PhD can contribute by converting prediction into usable decision support. |
| Retrospective design | Retrospective models can show association, but they are weaker for deployment and treatment recommendation. |
| Need prospective/RCT validation | For ovarian stimulation, dosing and trigger decisions, prospective evidence is important. |
| Single-center study | Single-center models often fail when moved to new clinics. This supports Indian validation as a meaningful gap. |
| Trustworthy/XAI gap | IVF decisions are high-stakes. Clinicians and patients need transparent explanations, not only model scores. |

## Medium Gaps

| Gap | Use Carefully |
| --- | --- |
| Small dataset | Useful only when repeated and when it affects generalization. |
| Limited population diversity | Strong when connected with Indian population or multi-clinic validation. |
| Lack of standardization | Useful for review contribution, but alone may be too broad for a PhD implementation topic. |
| Patient-centered XAI | Strong conceptual gap, but needs careful scope and measurable evaluation. |

## Gaps Not Enough Alone

These should not become the topic alone:

- one model had lower accuracy
- one paper used only one algorithm
- one paper did not use deep learning
- one paper did not include a specific feature

Those are limitations, not necessarily PhD-level gaps.
