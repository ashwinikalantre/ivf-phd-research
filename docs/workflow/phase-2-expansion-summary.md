## What Changed

The expansion strengthened the evidence base in three ways:

1. It added prospective/RCT evidence, not only retrospective prediction studies.
2. It added an Indian-population paper, directly relevant to the PhD context.
3. It added deployment/adoption and critical-review papers, which support trustworthy clinical decision-support framing.
