# Phase 4 Gap Summary

Phase 4 reduced the raw gap list into a PhD-ready gap shortlist.

## Final Gap Shortlist

| Gap | Role in Topic |
| --- | --- |
| External validation and generalizability | Methodological foundation |
| Clinical decision-support translation | Practical contribution |
| Trustworthy and explainable AI | Core informatics contribution |
| Multimodal personalization | Data/modeling contribution |
| Indian/contextual validation | Local relevance |
| Prospective or real-world usability evaluation | Clinical usefulness |

## What We Should Not Do

We should not make the final topic:

> Machine learning model for IVF success prediction.

That is too generic and already common.

## What We Should Do

The topic should focus on:

> Explainable and personalized clinical decision support for IVF outcome prediction.

## Strongest Working Title

**An Explainable and Personalized Clinical Decision Support Framework for IVF Outcome Prediction Using Multimodal Clinical and Embryological Data**

## Conditional Title Extensions

Use these only if data access supports them:

- `in Indian Women`
- `using Clinical, Embryological and Lifestyle Data`
- `with External Validation in Indian IVF Settings`
