## Main Finding

The literature does not support a title that is only:

> Machine learning model for IVF success prediction.

That would be too common.

The stronger direction is:

> Explainable, personalized and clinically usable IVF decision support.

## Best Topic Direction After Phase 3

**Explainable and personalized clinical decision support for IVF outcome prediction and treatment recommendation using multimodal clinical, embryology and contextual data.**

## Next Phase

Phase 4 should shortlist only the repeated gaps that can become a research problem:

- external validation
- clinical decision support
- trustworthy/XAI
- Indian/contextual validation
- multimodal clinical + embryology + lifestyle data
- real-world usability
