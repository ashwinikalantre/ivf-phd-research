# Review Protocol

## Inclusion Criteria

- AI, ML, DL, XAI or CDSS applied to IVF, ART, embryo selection, ovarian stimulation, FET, endometrial receptivity, live birth, pregnancy or miscarriage prediction.
- Published mainly from 2021 to 2025.
- Peer-reviewed journal preferred.
- Review papers allowed only for trend and gap synthesis.
- Preprints allowed only as supporting evidence, not final novelty proof.

## Exclusion Criteria

- Papers unrelated to IVF or ART.
- Purely biological papers without informatics, AI, ML or decision-support relevance.
- Non-academic clinic blogs or promotional sources.
- Papers with no usable abstract, method, dataset or limitation details.

## Extraction Fields

Each paper should be extracted into:

- paper number
- title
- authors
- year
- journal
- objective
- dataset
- features used
- method/algorithm
- technology used
- evaluation metrics
- main findings
- limitations
- future work
- gap found
- category
- relevance
- source URL
- extraction confidence

## Quality Flags

Use these confidence levels:

| Confidence | Meaning |
| --- | --- |
| High | Publisher/full text/PMC/PubMed verified and key fields extracted |
| Medium | Abstract and useful extraction available, but full limitations need checking |
| Low | Bibliographic or limitation details incomplete |

## Gap Rule

A gap becomes important only when it is:

- repeated across multiple papers
- relevant to IVF practice
- technically addressable by Computer Applications research
- feasible with obtainable data
- not already solved by a strong validated system
