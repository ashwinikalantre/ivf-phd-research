# Recommended Working Topic

## Why the Title Was Not Fixed Early

The title should come from literature gaps and data feasibility.

We first reviewed papers, identified repeated gaps, grouped themes, and checked what kind of data may be needed.

## Weak Topic Direction

This title is weak:

> Machine Learning Model for IVF Success Prediction

Reason:

Many papers already do this. It is too generic for a PhD.

## Strong Topic Direction

The stronger direction is:

> Explainable and personalized clinical decision support for IVF outcome prediction.

This direction is stronger because it addresses repeated gaps:

- explainability
- personalization
- clinical decision support
- validation
- multimodal data

## Recommended Working Title

**An Explainable and Personalized Clinical Decision Support Framework for IVF Outcome Prediction Using Multimodal Clinical and Embryological Data**

## If Indian Data Is Confirmed

**An Explainable and Personalized Clinical Decision Support Framework for IVF Outcome Prediction Using Multimodal Clinical and Embryological Data in Indian Women**

## If Lifestyle Data Is Confirmed

**An Explainable Multimodal Clinical Decision Support Framework for Personalized IVF Outcome Prediction Using Clinical, Embryological and Lifestyle Data**

## Final Decision Rule

Do not add words like `Indian`, `lifestyle`, or `external validation` in the final title unless the clinic data supports them.
