# Research Questions and Objectives

## Research Questions

**RQ1.** Which clinical, embryological, imaging, and contextual factors are useful for personalized IVF outcome prediction?

**RQ2.** Can a multimodal machine learning model predict IVF outcomes better than a model using only one type of data?

**RQ3.** How can explainable AI show the main reasons behind each prediction?

**RQ4.** Does the model work well for different patient groups, clinics, or Indian IVF data?

**RQ5.** How can prediction results be shown to doctors as useful decision support, without replacing the doctor?

## Objectives

1. To review recent AI, ML, XAI, and CDSS studies in IVF.
2. To identify repeated research gaps using a literature review matrix.
3. To design a multimodal IVF outcome prediction framework.
4. To use explainable AI for showing patient-specific prediction factors.
5. To evaluate model performance, calibration, and subgroup behavior.
6. To propose a clinician-facing decision support workflow for IVF counseling.

## What This Means

The PhD should not focus only on model accuracy.

It should focus on a complete decision support approach:

- prediction
- explanation
- validation
- clinical usability
