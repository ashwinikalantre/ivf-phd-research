# Faculty Presentation

This presentation summarizes the research journey from the broad IVF area to the current provisional PhD direction.

It is designed for discussion with university faculty. The working topic is intentionally presented as conditional on clinic data access.

## Presentation Structure

The deck covers:

- IVF and infertility context
- research-discovery process
- literature-review method
- six research themes
- repeated gap frequencies
- research problem
- provisional topic direction
- research questions
- conceptual framework
- dataset requirements
- technical blueprint
- expected contribution
- immediate next steps

## How To Use This Deck

Use the main slides for a 12-15 minute presentation.

Use the speaker script to keep the explanation simple and consistent.

Use the panel question bank after the presentation for discussion. It contains short answers, evidence points, cautions and questions to ask doctors.

## Important Presentation Position

The presentation does not claim that the final title, dataset or model has already been fixed.

The next decision depends on:

1. clinic data availability
2. outcome-variable quality
3. availability of embryological and lifestyle variables
4. multicentre or external-validation opportunities
5. clinician participation in explanation review
