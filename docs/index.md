# AI in IVF PhD Topic Discovery

This site presents the work done to discover a PhD topic in **AI for IVF**.

The topic was not selected first. It was developed from literature review, repeated gaps, and data feasibility.

## Current Working Title

**An Explainable and Personalized Clinical Decision Support Framework for IVF Outcome Prediction Using Multimodal Clinical and Embryological Data**

This is a working title. The final title depends on what data clinics can provide.

## Main Finding

Many papers already use machine learning for IVF success prediction.

So a topic like **“Machine Learning for IVF Success Prediction”** is too common.

The stronger PhD direction is:

> Explainable, personalized, and clinically usable decision support for IVF outcome prediction.

## Why This Direction Is Strong

The literature repeatedly shows these gaps:

- models are not externally validated
- prediction scores are not converted into clinical decision support
- many models are black-box
- Indian/context-specific validation is limited
- multimodal data are not consistently integrated
- real-world usability is weak

## What Must Be Confirmed Next

Before finalizing the title, we must confirm data access with doctors and clinics:

1. What IVF dataset can be accessed?
2. Does it include embryo variables?
3. Does it include lifestyle variables?
4. Is more than one clinic/source available?
5. Can clinicians review explanation outputs?
